# 仓库使用说明

## 项目简介

本项目是面向 3943B 采集链路的无人机射频信号预处理、数据集构建、模型训练与识别联调平台。

## 环境配置

- Python：建议 `3.10+`
- 核心依赖：`PySide6`、`numpy`、`scipy`、`opencv-python`、`torch`、`scikit-learn`
- 完整依赖安装：

```powershell
pip install -r requirements.txt
```

如果已有 CUDA 版 `torch` 环境，建议先保留原 CUDA 环境，再按需补装其他依赖，避免覆盖已有 GPU 版本。

## 使用步骤

### 第一步：模型放置

请将训练好的 `.pth` 模型文件放入 `/models` 文件夹，并在 `config.py` 中修改对应路径。
模型文件仅用于本地运行，不建议提交到 Git 仓库。

当前对应配置项：

```python
PREPROCESS_MODEL_PATH = PREPROCESS_MODELS_DIR / "best_model_1_detect_v2.pth"
```

### 第二步：启动程序

在仓库根目录执行：

```powershell
python main.py
```

### 第三步：按主流程联调

当前推荐按以下链路验证：

```text
信号预处理 -> 数据集管理 -> 生成数据集版本 -> 模型训练 -> 无人机识别
```

- 预处理页会自动扫描工作区根目录下全部 `.cap` 文件；输出候选样本后，会同步到数据集管理页。
- 数据采集页当前支持 `3943B` 联调模拟入口和 `USRP` 真实采集 V1；USRP 第一版输出独立 `.iq/.bin` 文件与同目录元数据 `.json`，不混入 CAP 算法主链路。
- 预处理页同时提供 `USRP IQ 演示预处理`，可把 B210 `.iq + .json` 切片成 `.npy` 样本，并同步登记到 `raw_files / preprocess_tasks / samples`，再进入数据集、训练和识别流程。
- USRP B210 调试步骤见 [USRP B210 Windows 连接与烟测说明](usrp_b210_setup.md)。当前 Windows 安装器可能因 PATH 过长无法自动写入环境变量，软件会自动查找 UHD 默认安装路径；当前已验通档位为 `1-5 Msps / 2 s`，软件默认用 `1 Msps / 2 s`。
- 南京演示建议按“真实采集 + 演示预处理桥接 + 训练识别”讲法：B210 展示现场采集和软件全流程，正式采集设备仍为 `3943B`；频点标签仅用于流程演示，不作为真实无人机型号。
- 数据集管理页负责手动标注、SQLite 持久化映射表自动标注、纳入/排除和数据集版本生成；当前 demo 建议统一使用 `类别A / 类别B / 类别C` 这类中性标签。
- 数据集划分策略已按真实语义执行：`按样本随机分层` 使用固定随机种子在标签内随机划分，`按设备个体隔离` 会保证同一设备不跨 train/val/test，设备数不足时会阻止生成。
- 生成数据集版本后，会输出 `data/datasets/{version_id}/manifest.json`。
- 训练页选择对应版本后，可执行数据检查、真实训练，并生成 `data/models/{model_id}/model.joblib` 与 `metadata.json`。
- 训练页内置“模型测试”工作区，可导入外部带标签 CSV（`sample_id,sample_file_path,label_type`）对真实模型做批量评估，并输出测试报告到 `data/evaluations/`。
- 识别页会直接读取数据库中的真实模型记录，对样本执行类型识别；若样本来源、中心频率或采样率超出训练适用域，会显示域外提示。
- 识别结果里的“预测置信度”是闭集模型最大类别概率，不是模型准确率；域外样本结果只能作为流程演示参考。
- 删除数据集版本时，不会删除已训练模型记录；清空样本数据库时，仍会一并清空模型记录。
- 数据采集页的 `移除记录（不删文件）` 只清数据库记录；信号预处理页的 `删除选中文件` 可选择同时删除本地原始文件；数据集页的 `清理孤立文件` 默认只预览数据库无引用的运行文件，确认后才删除。

`data/` 和 `db/` 都是本地运行数据目录，默认不提交到 Git 仓库。

## 提交前检查

无需 pytest 的基础 smoke 检查：

```powershell
python -B tools/smoke_checks.py
```

当前会覆盖 Python AST、数据集划分语义、识别页域外样本提示和 Qt `minimal` 主窗口实例化。

## 两分钟演示脚本

1. 展示“信号预处理”页的 CAP 文件、参数区和候选段结果。
2. 展示“数据集管理”页的样本入库、手动标注和映射表自动标注。
3. 演示“纳入数据集”勾选、生成数据集版本和 manifest。
4. 切到“模型训练”页，选择版本并点击“开始训练”。
5. 切到“无人机识别”页，选择刚生成的模型和一条样本，执行类型识别。
6. 简要说明删除样本、删除版本和清空样本数据库按钮用于联调清理，不建议演示时实际清空。
